# News-Spider
爬取国外新闻网站，并存储至Excel中

#### 特性
1. 支持界面操作
2. 支持时间段过滤
3. 支持多个关键词
4. 标题和正文附带谷歌机翻

#### 支持站点：
华尔街日报 + 福克斯新闻 + CNN + BBC + Olympic World + Olympic Tokyo + 国会山报 + Political

#### 技术关键词：
win10 + python 3.9 + requests + newpaper + selenium + pyside2

#### 注意事项：梯子自备
